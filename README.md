## Restyled Sonner Toast

Restyled the default Sonner toast from shadcn/ui with a few small CSS overrides to improve the appearance.

### Features

- Minimal CSS overrides
- Theme-aware (light / dark)
- Consistent variant colors
- Improved action and cancel button styles

### What changed

- Refined colors for the default toast
- Added a default icon for toasts without a type
- Cleaner toast button styles
- Better visual consistency across toast variants
- Improved light and dark theme support

### Toast variants

The following toast types are supported:

- success
- info
- warning
- error
- loading

Each variant has its own background, border, and text colors for better visual feedback.

### Why

The default Sonner styles are already good, but a few small tweaks can make
the toasts feel more consistent with the overall design system.

### Notes

This is only a visual restyle of the default Sonner toast used by shadcn/ui.  
No functionality or behavior from Sonner is modified.
